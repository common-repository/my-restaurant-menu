== Description ==
My Restaurant Menu plugin allows you to create online menu(s) for your restaurant, pizzeria, and so on. And you also get a printable PDF version, to upload to Facebook (if you have a Facebook page for your activity) or to hand to your guests.

Features:

* unlimited menu: lunch menu, dinner menu, summer menu, specials, wine list, etc.
* unlimited menus items: dishes, drink, etc.
* inside the admin area items are organized into different categories: starters, main course, etc.
* inside the menu items are organized into different sections, that can be the same as the 
* categories or completely different: catch of the day, special of the day, vegetarians, kids specials, etc.
* visual editor to create the menu, selecting items from the list of items you created
* items can be repeated in different sections of the menu: dish of the day also present in main courses, etc.
* featured image for items
* featured image for menus
* dedicated header and footer per menu
* icons for categories


== Installation ==
Use Wordpress' Add New Plugin feature, search for "My Restaurant Menu", or download the archive and:

1. Unzip the archive on your computer
2. Upload 'my-restaurant-menu' directory to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Create Menu Items and add them to Menus from the WordPress admin MRM menu
